# Basic Grading System
A beginner Python project that takes a student's marks and maximum possible marks, calculates the percentage, and outputs the grade and percentage.

# How to Run

1) Make sure Python is installed on your system
2) Download or clone the repository
3) Open terminal and navigate to the folder
4) Run the command: python grading_system.py
5) Enter your marks and maximum marks when prompted

# Grades:

A — above 90
B — 80 to 90
C — 60 to 80
D — 35 to 60
E — below 35 (repeat required)

# Features:

1) Handles decimal marks
2) Validates zero maximum marks
3) Displays rounded percentage alongside grad
